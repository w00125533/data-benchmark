# Superpowers Docs

Files under `docs/superpowers/specs/` and `docs/superpowers/plans/` are
historical planning and design records. They may contain commands or topology
descriptions that were valid when they were written, but they are not current
runbooks or operational startup instructions.

For current operational instructions, use the repository root `README.md` and
`../shared-data-infra/README.md`.

Current compose benchmark infrastructure is provided by `../shared-data-infra`.
`data-benchmark/docker-compose.yml` only owns `benchmark-runner`.
